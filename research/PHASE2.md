# Phase 2 — Full Torah Speech-Frame & Commandment Scan

## The question being resolved
Tomer's thesis: the fire prohibition (Exodus 35:3) is a rare/unique case of a commandment that appears ONLY in Moses's mouth ("וַיֹּאמֶר מֹשֶׁה... אֵלֶּה הַדְּבָרִים אֲשֶׁר צִוָּה ה'") with no direct divine-speech (A-frame) source for the specific content. Phase 1 (8 agents on secondary sources) confirmed: the fire prohibition IS textually unique in Exodus-Numbers (1 of 613), the question is respectable (Rashbam, Rihal, Philo, Temple Scroll, Heschel's R. Yishmael school all held the A/B distinction), but NO source concludes Moses authored it — all assume divine origin behind Moses's mouth.

Phase 2 reads the ACTUAL TEXT of the entire Torah, Deuteronomy INCLUDED, to build the definitive grid.

## Uniform classification schema (EVERY agent uses this)
For each commandment / law passage, produce a table row:
- **chapter:verse** (Hebrew book name)
- **speaker frame:**
  - **A** = direct divine speech. Narrator: "וַיְדַבֵּר/וַיֹּאמֶר יְהוָה אֶל מֹשֶׁה (לֵּאמֹר)" then content in God's voice. (Content inside a לֵאמֹר frame = A even though Moses relays it.)
  - **B** = Moses-attributed. Moses addresses the people: "וַיֹּאמֶר/וַיְדַבֵּר מֹשֶׁה... אֵלֶּה/זֶה הַדָּבָר אֲשֶׁר צִוָּה ה'" — content framed as God's command but spoken by Moses, with NO co-located A-frame.
  - **C** = narrative / narrator's voice / initiative, no divine-speech frame.
  - **D** = Deuteronomy (Moses's first-person retelling). Sub-classify EACH Deut law: does it have an A-frame parallel in Exodus-Numbers (mark D+A) or is it Deuteronomy-only (mark D-only)?
- **commandment content** (short Hebrew gloss)
- **permanence:** eternal/לדורות (♾) / Temple-or-Land-dependent (🏛) / one-time directive הוראת שעה (☄)
- **decisive test:** does this commandment have a direct A-frame source with the SPECIFIC content anywhere in the Torah? (yes / no / partial)

## Quote rule
Quote the opening frame verse and the commandment verse in Hebrew WITH nikud + chapter:verse. Use Sefaria (WebFetch sefaria.org/api/texts/...) for accurate text.

## Deliverable per agent
A Markdown table covering the assigned territory, then a bottom-line: count of A vs B vs C vs D, and a flagged list of every B-frame and every D-only commandment (these are the candidates that support or weaken Tomer's thesis).

## Manager (Opus) synthesis
After all territory agents finish, three cross-cutting agents build: (1) master speech-frame index, (2) closed Moses-only list across the whole Torah, (3) doublet map testing the kashrut/Shabbat "direct-then-repeated" claim. Then the manager produces the final verdict table.
